# Iris Flower Classification

## Project Overview
Machine learning classification of iris flower species.

## Objective
To classify iris flowers into species based on measurements.

## Dataset
Iris dataset with sepal and petal dimensions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Model Used
- Logistic Regression / KNN (any one you used)

## Key Insights
- Petal length is most important feature
- Model achieves high accuracy on dataset
